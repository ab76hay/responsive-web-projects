====================================================
        RESPONSIVE WEBPAGE PROJECT
====================================================

A responsive webpage built using HTML, CSS, and
Bootstrap. Designed to work smoothly across all
screen sizes including mobile, tablet, and desktop.

----------------------------------------------------
PROJECT STRUCTURE
----------------------------------------------------

  index.html         - Main HTML file (entry point)
  style.css          - Custom CSS styles
  /images            - Folder for images and assets
  /assets            - Folder for icons and other files
  README.txt         - Project documentation

----------------------------------------------------
FEATURES
----------------------------------------------------

  - Fully Responsive Design
    Works on all screen sizes (mobile, tablet, desktop)

  - Bootstrap Grid System
    Uses Bootstrap's 12-column grid for layout

  - Custom CSS Styling
    Additional styles on top of Bootstrap defaults

  - Navigation Bar
    Responsive navbar with hamburger menu on mobile

  - Clean & Modern Layout
    Simple, user-friendly interface design

----------------------------------------------------
TECHNOLOGIES USED
----------------------------------------------------

  - HTML5
  - CSS3
  - Bootstrap 5
  - Responsive Web Design principles

----------------------------------------------------
HOW TO RUN
----------------------------------------------------

  1. Download or clone the project folder

  2. Open the project folder on your computer

  3. Double-click on index.html
     OR
     Open it in any web browser (Chrome, Firefox, Edge)

  No server or installation required!

----------------------------------------------------
BOOTSTRAP SETUP
----------------------------------------------------

  Bootstrap is loaded via CDN. Make sure you have
  an internet connection, or download Bootstrap
  locally from: https://getbootstrap.com

  CDN links used in the project:

  CSS:
  https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css

  JS:
  https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js

----------------------------------------------------
RESPONSIVE BREAKPOINTS
----------------------------------------------------

  - xs  (< 576px)   - Extra small / Mobile
  - sm  (>= 576px)  - Small devices
  - md  (>= 768px)  - Tablets
  - lg  (>= 992px)  - Desktops
  - xl  (>= 1200px) - Large desktops

----------------------------------------------------
NOTES
----------------------------------------------------

  - Tested on Chrome, Firefox, and Edge browsers
